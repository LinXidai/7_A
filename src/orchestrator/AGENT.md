# AGENTS.md - 项目级 Agent 行为规则

## 安全规则
- 禁止执行 `rm -rf /` 等破坏性命令
- 涉及删除操作时必须先确认目标文件
- 不要操作 `.env` 文件，其中包含敏感信息

## 项目约定
- 本项目使用 Python 语言开发
- 优先使用项目虚拟环境中已安装的工具和库
- 代码文件统一使用 UTF-8 编码

## 语言偏好

- 优先用中文回复，除非对话历史让你用英文
- 回复简洁明了，避免冗余

## 交互偏好

- 优先且频繁使用 AskUserQuestion 工具与用户交互

## 写入文件规范

- 写入内容较多时，使用 AskUserQuestion 分批次确认后再写入

## 回答注意事项

- 写内容之前先经过用户同意
- 一点一点写，每部分写完后审阅
- 不要自行更新 Todos 内容，每完成一步先征求用户反馈
- 每次回复只回复用户让做的事情，做额外事情前先询问
